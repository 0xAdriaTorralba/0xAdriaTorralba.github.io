Talk given at **Web3 Family Conference 3** in Barcelona, April 2024.

## Abstract

This talk presents a systematic taxonomy for analysing the security of blockchain Layer 2 scalability solutions. As the blockchain ecosystem has grown, numerous Layer 2 approaches have emerged — rollups (optimistic and ZK), state channels, sidechains, and more — each with different security assumptions, trust models, and attack surfaces.

The talk classifies and examines these security assumptions and attack vectors across different L2 scaling approaches, providing a structured framework for researchers and practitioners to reason about the security guarantees (and limitations) of the systems they build on.
