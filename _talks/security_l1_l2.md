Talk given at the **Ethereum Dev Barcelona Meetup** in Barcelona, February 2024.

## Abstract

A practitioner-oriented presentation covering the security landscape of both Layer 1 blockchains and their Layer 2 extensions. The talk examines the security considerations developers and architects should keep in mind when building on or extending Ethereum's base layer and its growing L2 ecosystem.

Topics covered include the trust assumptions inherited from L1, the additional security surface introduced by L2 constructions, and practical guidance for reasoning about the security properties of different scaling approaches.
